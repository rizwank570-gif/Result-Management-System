Google Sheets Integration

Your application is configured to securely sync with Google Sheets.

Connection Settings

The Google Apps Script WebApp URL is now hardcoded directly in your application's source code for better security.

let cloudUrl = "YOUR_WEB_APP_URL";

How it works

Once connected via the source code, the system automatically performs two actions in your Google Sheet on every update:

  • It saves the raw backend data in a sheet named "SystemConfig".
  • It generates a beautifully formatted, readable table in a sheet named "MasterSheet" so you can directly view, print, or share from Excel/Google Sheets.
